# Changelog

This changelog tracks all changes that are visible in our API.

<!-- markdownlint-disable MD013 -->
## 2026-06-25

* **Added `same_party_type` comment visibility.**
  Comments marked `same_party_type` are visible to all involved parties of the
  same type as the creator (e.g. a system operator's comment can be read by
  other system operators involved in the same resource, but not by service
  providers). This applies to all comment resources.

## 2026-06-08 · Initial release

* **Initial public release of the Flexibility Information System API.**
  All endpoints are available under `Api-Version: 2026-06-08`.
